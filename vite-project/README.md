# Winery Landing

A modern, cinematic landing page for a winery featuring a film noir aesthetic with interactive wine cards, smooth animations, and an elegant user experience.

## Features

- **Cinematic Design**: Film noir-inspired UI with vintage film effects
- **Interactive Wine Cards**: Flip cards showcasing wine collections with detailed information
- **Smooth Animations**: Hero text effects, scroll animations, and interactive elements
- **Responsive Design**: Mobile-first approach with optimized layouts
- **Shopping Cart**: Integrated cart functionality for wine reservations

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- React Leaflet (for maps)

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
